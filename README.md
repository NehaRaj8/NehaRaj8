##  Featured project — Patient Condition Prediction API

**End-to-end ML application: from training to containerised REST API**

```
Clinical data → Scikit-learn classifier → FastAPI REST API → Docker container → Live predictions
```

- Supervised classification model trained on clinical indicators (age, heart rate, systolic BP)
- REST API with Pydantic validation, confidence scoring, and full Swagger/OpenAPI docs
- Containerised with Docker following MLOps best practices
- Model serialised with Joblib for reproducible, production-ready deployment

**Tech:** `Python` `Scikit-learn` `Pandas` `FastAPI` `Docker` `Uvicorn` `Pydantic` `Joblib`
